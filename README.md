# Pagami Motors Static Website

## Overview
Pagami Motors is a responsive static website developed to showcase vehicle-related services, company information, and contact details in a professional and user-friendly manner.

This project demonstrates the use of modern web development technologies to create an attractive and functional website.

## Features
- Responsive design for mobile, tablet, and desktop devices
- Clean and modern user interface
- Home page with company introduction
- Services section
- About Us section
- Contact information page
- Easy navigation

## Technologies Used
- HTML5
- CSS3
- JavaScript
